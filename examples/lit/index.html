🎬 Lottie Lit (Web Components) 示例

使用标准 Web Components API，无需框架！

1️⃣ 基础 Web Component

使用 <lottie-animation> 标签即可！
💡 优势:
  • 无需任何框架
  • 标准 Web API
  • 可在任何项目中使用
  • 浏览器原生支持
<lottie-animation src="../assets/loading.json" loop="true" autoplay="true" renderer="svg" ></lottie-animation>

2️⃣ 播放器 Component (带控制栏)

内置完整的播放控制栏
<lottie-player src="../assets/heart.json" loop="true" controls="true" ></lottie-player>

3️⃣ JavaScript 控制

const element = document.getElementById('js-controlled'); // 调用方法 element.play(); element.pause(); element.stop(); element.reset(); element.setSpeed(2);

4️⃣ 事件监听

等待事件触发...
const element = document.getElementById('event-animation'); element.addEventListener('ready', () => { console.log('动画加载完成'); }); element.addEventListener('complete', () => { console.log('动画播放完成'); }); element.addEventListener('error', (e) => { console.error('错误:', e.detail); });

5️⃣ 动态创建元素

const lottie = document.createElement('lottie-animation'); lottie.setAttribute('src', '../assets/loading.json'); lottie.setAttribute('loop', 'true'); lottie.setAttribute('autoplay', 'true'); document.getElementById('container').appendChild(lottie);

6️⃣ 多个实例

多个 Web Components 可以共存，完全独立运行

✨ Web Components 的优势

🌐 框架无关

可在任何框架中使用，甚至纯 HTML

📦 封装性好

Shadow DOM 隔离，不会污染全局样式

🔧 易于使用

像使用普通 HTML 标签一样简单

♻️ 可复用

一次定义，到处使用
