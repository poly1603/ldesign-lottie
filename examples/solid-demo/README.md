# Solid.js Lottie Demo

Demo showcasing `@ldesign/lottie-solid` with fine-grained reactivity.

## Features

- ✅ Component API
- ✅ Composable Hook (useLottie)
- ✅ Reactive speed control
- ✅ Event handling
- ✅ Minimal bundle size

## Getting Started

```bash
pnpm install
pnpm dev
```

Open http://localhost:3001

## Usage Examples

See `src/App.tsx` for:
- Component usage
- useLottie composable
- Signal-based reactivity
- Event handlers
