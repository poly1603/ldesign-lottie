🎬 Lottie 全框架使用指南

Vue 3 · React · Web Components · Vanilla JS - 四种方式，任您选择

💚

Vue 3

📦 Composable 用法 推荐
import { useLottie } from '@ldesign/lottie/vue' const { play, pause } = useLottie({ container: ref(null), path: '/animation.json' })
🎨 组件用法
<!-- LottieAnimation --> <LottieAnimation path="/animation.json" :loop="true" /> <!-- LottiePlayer (带控制栏) --> <LottiePlayer path="/animation.json" :showControls="true" />
✨ 指令用法
<!-- 基础指令 --> <div v-lottie="/animation.json" /> <!-- 悬停指令 --> <div v-lottie-hover="/animation.json" /> <!-- 滚动指令 --> <div v-lottie-scroll="/animation.json" />
✓ 3 个 Composables
✓ 3 个组件
✓ 3 个指令
✓ 全局插件支持
⚛️

React

🪝 Hook 用法 推荐
import { useLottie } from '@ldesign/lottie/react' const { containerRef, play, pause } = useLottie({ path: '/animation.json', loop: true }) return <div ref={containerRef} />
🎨 组件用法
// LottieAnimation <LottieAnimation path="/animation.json" loop={true} autoplay={true} /> // LottiePlayer (带控制栏) <LottiePlayer path="/animation.json" showControls={true} />
🔗 Context 用法
<LottieProvider> <YourApp /> </LottieProvider> // 在子组件中 const { playAll, pauseAll } = useLottieContext()
✓ 4 个 Hooks
✓ 3 个组件
✓ Context Provider
✓ TypeScript 完整支持
🌐

Web Components

🏷️ 标签用法 框架无关
<!-- 基础组件 --> <lottie-animation src="/animation.json" loop="true" autoplay="true" ></lottie-animation> <!-- 播放器组件 --> <lottie-player src="/animation.json" controls="true" ></lottie-player>
⚡ JavaScript 控制
const element = document.getElementById('my-lottie') // 调用方法 element.play() element.pause() element.setSpeed(2) // 监听事件 element.addEventListener('ready', () => { console.log('Ready!') })
✓ 标准 Web API
✓ 无需框架
✓ Shadow DOM 隔离
✓ 可在任何地方使用
⚡

Vanilla JS

🎯 纯 JavaScript 最灵活
import { createLottie } from '@ldesign/lottie' const animation = createLottie({ container: '#lottie', path: '/animation.json', loop: true, autoplay: true }) animation.play()
🎮 高级功能
import { AnimationSequence, InteractiveController, VirtualRenderer } from '@ldesign/lottie' // 序列播放 const seq = new AnimationSequence() // 交互控制 new InteractiveController({ instance })
✓ 完整 API 访问
✓ 最大灵活性
✓ 所有高级功能
✓ 性能优化工具

💡 如何选择？

选择 Vue 3

  • Vue 3 项目
  • 需要指令简化代码
  • 喜欢 Composition API
  • 需要全局插件管理

选择 React

  • React 项目
  • 喜欢 Hooks 模式
  • 需要 Context 全局管理
  • TypeScript + React 项目

选择 Web Components

  • 多框架项目
  • 纯 HTML 页面
  • 需要最大兼容性
  • 微前端架构

选择 Vanilla JS

  • 需要完全控制
  • 复杂的自定义需求
  • 使用所有高级功能
  • 性能极致优化
