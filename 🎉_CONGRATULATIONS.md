## 📚 快速开始

### 1分钟开始使用

```html
<!-- 最简单：Web Components -->
<lottie-animation src="/animation.json"></lottie-animation>
```

```vue
<!-- Vue 3：一行代码 -->
<div v-lottie="'/animation.json'" />
```

```tsx
// React：一行代码
<LottieAnimation path="/animation.json" />
```

### 5分钟启用优化

```typescript
import {
  VirtualRenderer,      // 虚拟化渲染
  memoryManager,        // 内存管理
  workerManager         // Worker 加速
} from '@ldesign/lottie'

// 大量动画？
new VirtualRenderer().register(animation)

// 长期运行？
memoryManager.startMonitoring()

// 大文件？
await workerManager.parseAnimation(data)
```

## 🎯 使用建议

### 根据项目选择

```
Vue 3 项目
└─ 推荐: Vue Composable 或组件或指令
   └─ 最简单: <div v-lottie="..." />

React 项目  
└─ 推荐: React Hook 或组件
   └─ 最简单: <LottieAnimation path="..." />

多框架项目
└─ 推荐: Web Components
   └─ 最简单: <lottie-animation src="..." />

需要完全控制
└─ 推荐: Vanilla JS + 所有优化
   └─ 最强大: 完整 API 访问
```
